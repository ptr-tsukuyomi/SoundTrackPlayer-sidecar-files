# SoundTrackPlayer(仮称)用ループ設定済みサイドカーファイル

## 概要
[SoundTrackPlayer(仮称)](https://github.com/ptr-tsukuyomi/SoundTrackPlayer)で楽曲をループ再生するために使用するファイルです。
CD1枚分ぐらいの単位でまとめて格納しています。

## 使い方
このリポジトリにお手持ちのCD用のファイルがあれば、以下のようにして使用できます。

1. お手持ちのCDをリッピングしてどこかのフォルダに入れます。
2. このリポジトリからファイルをダウンロードして、1. のフォルダに移動します。
	- ファイル名の末尾が`.toml` になっているファイル以外はコピー不要です。削除して問題ありません。
3. 1.のフォルダ内で、2.のファイル名を `<1.でリッピングしたファイル名>.toml` に変更します。
	- (例) 1.で`01.flac`という名前のファイルを作成した場合、2.のファイル名を`01.flac.toml`に変更します。
4. SoundTrackPlayer(仮称)にて、1.でリッピングしたファイルを再生キューに追加するなどして読み込み、再生します。
